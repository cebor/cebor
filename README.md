## 👨‍💻 About Me

I'm an **open source developer with 20+ years of experience** and a deep love for detail. I write clean, maintainable code and always stay on the bleeding edge of technology. My current passion is integrating AI into dev workflows using local models.

- 🧹 **Clean Code** — every line matters, no shortcuts taken
- 🔭 **New Technologies** — always exploring what's next
- 🤖 **Local AI** — building and coding alongside local LLMs via [Ollama](https://ollama.ai)
- 🦀 **Rust enthusiast** — because performance and safety aren't a tradeoff
- 🏠 **Home Lab** — running Kubernetes, Home Assistant & self-hosted infra at home
